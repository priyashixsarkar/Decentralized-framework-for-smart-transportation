Decentralized Framework of Smart Transportation System to Achieve Secured Data Management Using Blockchain and Cognitive AI Based Firewall

Project Overview

This system integrates a **Cognitive AI Firewall** (trained using a Random Forest Model at 93.8% accuracy) to intercept all incoming user ticket booking requests.

By processing behavioral vectors such as `login hour`, `time spent`, `device/location anomalies`, and `failed attempts`, the system instantly isolates and blocks malicious traffic. Finally, it stores all interactions (both successful and blocked) on an immutable **Ethereum Web3 Smart Contract Blockchain** ledger.
